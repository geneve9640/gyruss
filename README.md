# Gyruss

This fork is specifically for the Geneve 9640 computer

Ported in 2026 from the Colecovision version by Rasmus Moustgaard <rasmus.moustgaard@gmail.com>. 

## Requirements

* Geneve 9640 computer
* MDOS operating system, 7.40 or higher

## Loading the game with GPL

* Enable TIMODE via the Autoexec startup
* load GPL from the command line
* load the editor assembler cartridge or use 80 column menu
* load GYRUSS1

## Loading the game with EXEC
* EXEC GYRUSS1
* Specify the correct path for the game
* Specify the correct path for EXEC or have in yout search path
* Example: C:\EXEC TIP1.GYR.GYRUSS1

## Playing the game

Joystick only

SPACE - pause

ESCape - quits the game

You can activate two-player mode by starting the game by pressing the fire button on joystick 2.
