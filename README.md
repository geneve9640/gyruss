# Gyruss

For the TI-99/4A home computer.

Ported in 2026 from the Colecovision version by Rasmus Moustgaard <rasmus.moustgaard@gmail.com>. 

## Requirements

* 32K memory expansion card.
* Disk drive, FinalGROM, physical cartridge, or similar.

## Loading the game

* From cartridge or FinalGROM
* From disk and XB cartridge, use XB autorun
* From disk and E/A cartridge, load DSK1.GYRUSS from option 5

## Playing the game

Joystick only.

P - Pause

You can activate two-player mode by starting the game by pressing the fire button on joystick 2.
