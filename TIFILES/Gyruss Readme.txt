Gyruss
This fork is specifically for the Geneve 9640 computer

Ported in 2026 from the Colecovision version by Rasmus Moustgaard rasmus.moustgaard@gmail.com.  https://github.com/Rasmus-M/gyruss


Requirements
Geneve 9640 computer
MDOS operating system, 7.40 or higher
Loading the game
This game requires the use of the GYRLOAD loader Example: C:\GYRLOAD TIP1.GYR.GYRUSS1

Playing the game
Joystick only

P - pause

ESCape - exits the game

You can activate two-player mode by starting the game by pressing the fire button on joystick 2.
